# Student Portfolio — Erin M. Quiazon

A responsive personal portfolio website built with **Bootstrap 5.3** via CDN.

## Tech Stack
- HTML5 (semantic elements)
- Bootstrap 5.3 (CDN — no npm/build step)
- CSS Custom Properties (Bootstrap token overrides)
- Vanilla JavaScript
- Google Fonts (DM Serif Display + Jost + DM Mono)
- Bootstrap Icons

## Bootstrap 5 Features
1. **Navbar** — `.navbar-expand-lg` with fixed positioning and hamburger collapse
2. **Cards** — Projects, interests, experience, education, references, coursework
3. **Modals** — Click project cards for full details with `data-bs-toggle="modal"`
4. **Badges** — Status (Completed/In Progress), tech stack, skill tags
5. **Grid System** — `.container`, `.row`, `.col-*`, `.row-cols-*` throughout
6. **Tooltips** — `data-bs-toggle="tooltip"` on social icons and skill tags

## Sections
1. Navbar (fixed, responsive, dark mode toggle)
2. Hero (full-viewport, two-column layout)
3. Stats (4 quick stats)
4. Clock (live time + greeting)
5. About (bio, skills, interests, goals)
6. What I Do (4 interest cards)
7. Projects (3 cards with modals)
8. Skills (progress bars with ARIA)
9. Experience (3 timeline cards)
10. Education (DLSU + Holy Angel + coursework grid)
11. References (3 reference cards)
12. Contact (validated form + social links)
13. Footer

## Deployment
Deployed on [Vercel](https://vercel.com) via GitHub integration.
